eappi.policy Installation
-------------------------

To install eappi.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``eappi.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        eappi.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
